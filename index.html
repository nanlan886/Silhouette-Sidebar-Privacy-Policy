侧影-侧边栏是一款本地运行的侧边栏工具类应用。我们非常重视你的隐私和个人信息保护。本应用当前版本不提供账号登录、不接入广告、不接入统计分析 SDK、不接入推送 SDK、不提供云同步功能，也不会将你的数据上传至服务器。

本政策将向你说明本应用在使用过程中涉及的权限、数据处理方式以及你的相关权利。

一、本应用是否联网

当前版本为本地运行应用，不需要联网即可使用主要功能。

本应用不会主动向服务器上传你的个人信息、应用列表、使用记录、配置数据或其他本地数据。

如后续版本增加联网功能、第三方 SDK、云同步、广告、统计或账号服务，我们会更新本隐私政策，并在必要时重新取得你的同意。

二、我们处理的信息和使用目的

为实现侧边栏功能，本应用可能会在你的设备本地处理以下信息：

1. 侧边栏配置数据

包括侧边栏位置、大小、透明度、显示样式、快捷入口、功能开关、游戏模式白名单等。

使用目的：保存你的个性化设置，保证侧边栏按你的配置运行。

处理方式：仅保存在你的设备本地，不上传服务器。

2. 已安装应用列表

包括应用名称、应用图标、应用包名。

使用目的：用于在设置页面中展示可添加到侧边栏的应用，以及用于设置游戏模式白名单。

处理方式：仅在本地读取和展示，不上传服务器，不用于用户画像或其他无关用途。

3. 应用使用情况信息

当你开启“使用情况访问权限”后，本应用可能读取当前或近期正在使用的应用信息。

使用目的：用于判断你当前是否打开了游戏模式白名单内的应用，从而自动执行与游戏模式相关的本地功能。

处理方式：仅在你的设备本地进行判断，不上传服务器，不保存你的详细应用使用记录，不分析你的使用习惯。

4. 运行状态与本地设置

包括应用是否开启、侧边栏是否显示、相关功能是否启用等状态。

使用目的：保证侧边栏持续运行，并在系统清理后台后尽量恢复正常工作。

处理方式：仅在本地使用。

三、本应用申请的系统权限说明

1. 悬浮窗权限

权限用途：用于在其他应用界面上方显示侧边栏入口或侧边栏面板。

如果你关闭该权限，侧边栏将无法正常显示。

2. 使用情况访问权限

权限用途：用于检测当前打开的应用是否属于你设置的游戏模式白名单。

本应用不会通过该权限上传、出售或共享你的应用使用记录。

如果你关闭该权限，游戏模式自动识别功能可能无法正常使用。

3. 读取应用列表

权限用途：用于让你选择哪些应用加入侧边栏快捷入口，或加入游戏模式白名单。

本应用仅读取应用名称、图标和包名，并仅在本地使用。

如果你关闭或限制该能力，相关应用选择功能可能无法正常使用。

4. 自启动设置

本应用可能引导你开启自启动权限。

用途是让侧边栏在设备重启后或被系统清理后可以继续运行。

该设置不用于收集个人信息。

5. 省电无限制/后台运行设置

本应用可能引导你将应用设置为省电无限制或允许后台运行。

用途是减少系统对侧边栏服务的后台限制，保证侧边栏持续可用。

该设置不用于收集个人信息。

6. 关联启动权限

部分手机系统可能提供“关联启动”或类似权限设置。本应用可能引导你开启该权限。

用途是保证你通过侧边栏打开已添加的应用时，相关快捷启动功能可以正常执行；同时在应用被系统限制或清理后，帮助侧边栏相关服务恢复运行。

本应用不会利用关联启动权限擅自启动无关应用，不会在后台频繁拉起其他应用，也不会通过该权限收集、上传或共享你的个人信息。

四、第三方 SDK

当前版本不接入第三方 SDK，包括但不限于广告 SDK、统计 SDK、推送 SDK、支付 SDK、地图 SDK、登录 SDK 等。

如后续接入第三方 SDK，我们会在隐私政策中说明 SDK 名称、提供方、使用目的、收集信息类型及第三方隐私政策链接。

五、信息存储

本应用产生的配置数据主要保存在你的设备本地。

当你卸载本应用或清除应用数据后，本地配置数据通常会被删除，具体以你的设备系统规则为准。

六、信息共享、转让和公开披露

当前版本不联网，不会向服务器上传数据，也不会向第三方共享、转让或公开披露你的个人信息。

但在以下情况下，我们可能依法处理相关信息：

1. 根据法律法规、司法机关或行政机关要求。

2. 为保护用户、开发者或公众的合法权益所必需。

3. 获得你的明确同意。

七、你的权利

你可以通过以下方式管理你的信息和权限：

1. 在系统设置中关闭悬浮窗权限。

2. 在系统设置中关闭使用情况访问权限。

3. 在系统设置中取消自启动、省电无限制或后台运行设置。

4. 在应用内删除侧边栏快捷入口、游戏模式白名单等配置。

5. 通过系统设置清除本应用数据。

6. 卸载本应用。

关闭部分权限后，对应功能可能无法正常使用，但不会影响其他不依赖该权限的功能。

八、未成年人保护

本应用为工具类应用，不主动面向未成年人收集个人信息。未成年人使用本应用时，应在监护人指导下使用。

九、政策更新

如果本应用功能、权限、数据处理方式发生变化，我们会更新本隐私政策。涉及重要变化时，我们会通过弹窗、页面提示或其他合理方式提醒你阅读。
